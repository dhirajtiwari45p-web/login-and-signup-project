Login & Signup Form

📌 Project Description

This is a simple Login and Signup form created using HTML, CSS, and JavaScript.
Users can create a new account using the signup form and then log in using their credentials.

🚀 Features

- User Signup Form
- User Login Form
- Form Validation using JavaScript
- Responsive Design
- Simple and clean UI

🛠️ Technologies Used

- HTML
- CSS
- JavaScript

📂 Project Structure

Login-Signup-Project
│── index.html
│── style.css
│── script.js
│── README.md

▶️ How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Open index.html in your browser.
